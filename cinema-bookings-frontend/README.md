# Cinema Bookings Frontend

This project is a cinema bookings application that allows users to browse movies, select seats, and finalize their bookings. The application is built using HTML, CSS, and JavaScript.

## Project Structure

```
cinema-bookings-frontend
├── src
│   ├── index.html            # Main entry point for the application
│   ├── pages
│   │   ├── home.html         # Home page displaying featured movies
│   │   ├── movie-details.html # Detailed view of a specific movie
│   │   ├── checkout.html      # Review selected movies and seats
│   │   └── confirmation.html   # Confirmation page after booking
│   ├── components
│   │   ├── header.html        # Header component with navigation
│   │   ├── footer.html        # Footer component with copyright info
│   │   ├── movie-card.html    # Structure for individual movie cards
│   │   ├── seat-map.html      # Layout for seat selection
│   │   └── booking-form.html   # Booking form for user details
│   ├── css
│   │   ├── styles.css         # Main styles for the application
│   │   ├── layout.css         # Layout-specific styles
│   │   └── components.css     # Component-specific styles
│   └── js
│       └── main.js            # Main JavaScript file for interactivity
└── README.md                  # Project documentation
```

## Features

- Browse and view details of movies
- Select seats for booking
- Review and confirm bookings
- Responsive design for various devices

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd cinema-bookings-frontend
   ```

3. Open the `src/index.html` file in your web browser to view the application.

## Usage Guidelines

- Use the navigation links in the header to explore different pages.
- On the home page, you can view featured movies and click on them for more details.
- Follow the prompts to select seats and complete your booking.
